# Project Idea Template
This [Trilium](https://triliumnotes.org/) template can be used to quickly document and evaluate project or new feature ideas.

## How to Use in Trilium
1. Download the HTML file from this repo.
1. Choose where you want to store your template.  This could be in the root or in a specific template collection.
1. Right-click and select 'Import into Note'.
1. Select the downloaded template HTML file.  You can leave all other options as default.
1. Click 'Import'.  This will create a new note.
1. Open the newly created note.
1. Set the note as a template.  This option can be found in the ... menu at the top right of the editing pane.  Set the 'Template' option.
1. Change the title and icon if desired.

## Creating a New Note from Template
1. Right-click on the parent node where you would like to create an instance of the template.
1. Hover over 'Insert child node'.
1. Your project templates should show in the menu to the right.

For more information about using templates in Trilium notes see [the docs](https://docs.triliumnotes.org/user-guide/advanced-usage/template).
